UX Design

UX designer skills;
  1. Useful - original and fulfill needs
  2. Usable - easy & enjoyable to use, easy to navigate
  3. Desirable - not UI design or graphic design but brings both together
  4. Learn-able- user learns whilst using
  5. Memorable- familiarity and can pick up where left off
  6. Accessible - accessible to all regardless of ability or any disability
  7. Credible - Trust and believe in product
UX design is its own field and does overlap into areas pertaining to Graphic design, Visual Design or UI design but it does cover the following;
UX designer skills;
  • Wireframes - rough guide, about structure of map leading the user
  • User research - smart decisions about demographics, asking questions
  • Desirable - not UI design or graphic design but brings both together
  • Copywriting - is it clear? is it functional? does it work?
  • User Persona - creation of the specific user, a representation of the core demographics
  • User Testing- design , create and use - improve then design, create and use .. . .
